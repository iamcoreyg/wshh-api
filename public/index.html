(Unoffical) WorldStarHipHop API

To use this api, send a GET request to:

wshh-api.herokuapp.com/video/VIDEO_ID_HERE

Currently, you're only able to get information about a single video. Expect a JSON response that provides the title, video url, number of views, and upload date like:


{
"status":"ok",
"title":"Throwback Of The Week: A Young Vin Diesel Break Dancing In The 80's",
"video_url":"http://www.worldstarhiphop.com/videos/video.php?v=wshhWj74x1979C13t0q4",
"views":"407,862 views",
"date":"Uploaded July 31, 2014"
}
